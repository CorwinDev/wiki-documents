## New Wiki Added

### Audio Library For Wio Terminal using ReSpeaker 2-Mic Hat

[This wiki](https://wiki.seeedstudio.com/Wio-Terminal-Audio-Play-Record/) introduces how to use the Audio library with [Wio Terminal](https://www.seeedstudio.com/Wio-Terminal-p-4509.html) using [ReSpeaker 2-Mic Hat](https://www.seeedstudio.com/ReSpeaker-2-Mics-Pi-HAT.html). With the help of this library, you are able to record, play and analysis audio using Wio Terminal! Furthermore, this audio library can be implemented into different areas such as Speech recognition and much more!
