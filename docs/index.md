### 1. Bluetooth Functionality on the Wio Terminal

We have prepared a [wiki](https://wiki.seeedstudio.com/Wio-Terminal-Bluetooth-Overview/) to introduce how to update the Bluetooth firmware for the Realtek RTL8720 Wireless Core on the [Wio Terminal](https://www.seeedstudio.com/Wio-Terminal-p-4509.html), as well as installing all the dependent libraries for Wio Terminal to enable BLE functionality.
Also, if you want to connect to other bluetooth devices using the Wio Terminal, you can follow [this wiki](https://wiki.seeedstudio.com/Wio-terminal-BLE-application/) to get started!

### 2. Ethernet on Wio Terminal using ENC28J60 OVERLAYS HAT for Raspberry Pi

If you ever wanted a stable ethernet connection with your Wio Terminal, now it's all possible with the [ENC28J60 OVERLAYS HAT for Raspberry Pi](https://www.seeedstudio.com/ENC28J60-OVERLAYS-HAT-for-Raspberry-pi-p-3045.html) connected to the Wio Terminal. You can follow [this guide](https://wiki.seeedstudio.com/Wio-Terminal-Ethernet/) to get started! This is a perfect example of using Raspberry Pi 40 Pin extensions board on Wio Terminal.

<p style="text-align:center;"><a href="https://wiki.seeedstudio.com/Wio-Terminal-Ethernet/"><img src="https://files.seeedstudio.com/wiki/Wio-Terminal-Ethernet/twitter.png" alt="pir"  width="900" height="auto"></a></p>

### 3. Use High Precision Barometric Pressure Sensor with Wio terminal

We have prepared a [step-by-step guide](https://wiki.seeedstudio.com/Barometric-Pressure-sensor-using-the-Wio-terminal/) to explain you, how to use the [Grove- high precision barometric pressure sensor (DPS310)](https://www.seeedstudio.com/Grove-High-Precision-Barometer-Sensor-DPS310-p-4397.html) with the Wio Terminal. This allows you to take high precision pressure sensor measurements and display the results on an analog-style meter!
