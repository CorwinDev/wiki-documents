### 2. CircuitPython on Wio Terminal

[CircuitPython](https://circuitpython.org/) is a programming language designed to simplify experimenting and learning to program on low-cost microcontroller boards. It makes getting started easier than ever before with no upfront desktop downloads needed. Once you get your board set up, open any text editor, and get started editing code.
Now you can install and run the official CircuitPython by Adafruit Industries on the Wio Terminal by following [this guide](https://wiki.seeedstudio.com/Wio-Terminal-CircuitPython/)!

<p style="text-align:center;"><a href="https://wiki.seeedstudio.com/Wio-Terminal-CircuitPython/"><img src="https://files.seeedstudio.com/wiki/Wio-Terminal-CircuitPython/cp-wt.png" alt="pir"  width="900" height="auto"></a></p>
