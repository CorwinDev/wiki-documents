## New Wiki Added

### reTerminal Buildroot SDK

[Buildroot](https://buildroot.org/) is an easy-to-use tool that simplifies and automates the process of building a complete Linux system for an embedded system, using cross-compilation.

We have prepared a [step-by-step guide](https://wiki.seeedstudio.com/reTerminal-Buildroot-SDK) to build your own [reTerminal](https://www.seeedstudio.com/ReTerminal-with-CM4-p-4904.html) system image using Buildroot.

### reTerminal Yocto SDK

The [Yocto Project](https://www.yoctoproject.org/) is an open source collaboration project that helps developers create custom Linux-based systems for embedded products, regardless of the hardware architecture.

We have prepared a [step-by-step guide](https://wiki.seeedstudio.com/reTerminal-Yocto) to build your own reTerminal system image using Yocto.

### Grove - Temperature & Humidity Sensor (DHT20)

The new Grove - Temperature & Humidity Sensor is based on the DHT20 sensor. The DHT20 is an upgraded version of the DHT11, compared with the previous version, the temperature and humidity measurement accuracy are higher, and the measurement range is larger. It features I2C output which means it is easier to use.

Click [here](https://wiki.seeedstudio.com/Grove-Temperature-Humidity-Sensor-DH20) to get started!

## Existing Wiki Updated

### Build Your Own GUI Application Using LVGL

LVGL provides everything you need to create an embedded GUI with easy-to-use graphical elements, beautiful visual effects and a low memory footprint. LVGL is a graphics library targeting microcontrollers with limited resources. 

We have updated the [wiki guide](https://wiki.seeedstudio.com/reTerminal-build-UI-using-LVGL/#build-your-own-gui-application) to help you build your own GUI application using LVGL

### reTerminal Raspberry Pi OS Latest Image

We have released the latest Raspberry Pi OS image for the reTerminal which fixes a major bug where the LCD didn't work after the first update. Please [click here](https://wiki.seeedstudio.com/reTerminal-FAQ/#q2-how-can-i-flash-raspberry-pi-os-which-is-originally-shipped-with-reterminal) to download the files for 32-bit and 64-bit, and also [check here](https://wiki.seeedstudio.com/reTerminal/#flash-raspberry-pi-os-64-bit-ubuntu-os-or-other-os-to-emmc) to learn how to flash the OS to the onboard eMMC.
